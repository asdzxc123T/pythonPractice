OracleDB
    일반적인 데이터 보관
    -> 보안이 중요
    -> 관리자 계정, 유저 계정, 권한
    -> 규칙을 정의하고 규칙에 맞게 데이터를 관리
    OracleDB > DB > TableSpace > Table > Data
               xe   ljwTS        apr07..
    SQL로 제어

MongoDB
    비정형데이터 보관
    -> 보안이 안 중요해도 됨
    -> 계정 시스템이 있기는 한데...
    -> 정의하지 않음, 처음 썼을 때 없으면 자동 생성
    MongoDB > DB > JS배열 > JS객체
              test 
    JavaScript(PL, Python과 문법이 비슷)로 제어
    조회 결과 : JS배열 + JS객체 -> JSON -> Python list + dict

자기 이름으로 된 DB 만들자 - 없음
그쪽으로 연결하자 - 
    use DB명
    db.JS배열.insert(JS객체);
    db.JS배열.find();
